## Usage

    kubes logs

## Description

logs from all deployment pods


## Options

```
    [--compile], [--no-compile]  # whether or not to compile the .kube/resources
                                 # Default: true
n, [--name=NAME]                 # deployment name to use. IE: demo-web
f, [--follow], [--no-follow]     # Follow logs
                                 # Default: true
    [--verbose], [--no-verbose]  
    [--noop], [--no-noop]        
```
