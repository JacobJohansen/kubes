## Usage

    kubes delete [APP] [RESOURCE]

## Description

Delete Kubernetes resources within the app folder


## Options

```
    [--image=IMAGE]              # override image
y, [--yes], [--no-yes]           # Skip are you sure prompt
    [--verbose], [--no-verbose]  
    [--noop], [--no-noop]        
```
