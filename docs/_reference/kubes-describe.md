## Usage

    kubes describe [ROLE] [RESOURCE]

## Description

Describe Kubernetes resource using the compiled YAML files


## Options

```
[--image=IMAGE]              # override image
[--compile], [--no-compile]  # whether or not to compile the .kube/resources
                             # Default: true
[--verbose], [--no-verbose]  
[--noop], [--no-noop]        
```
