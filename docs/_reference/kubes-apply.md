## Usage

    kubes apply [APP] [RESOURCE]

## Description

Apply the Kubenetes YAML files without changing them


## Options

```
[--image=IMAGE]              # override image
[--compile], [--no-compile]  # whether or not to compile the .kube/resources
                             # Default: true
[--verbose], [--no-verbose]  
[--noop], [--no-noop]        
```
